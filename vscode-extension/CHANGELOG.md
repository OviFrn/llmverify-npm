# Changelog

All notable changes to the llmverify VS Code extension.

## [1.0.0] - 2024-12-04

### Added
- Initial release
- Verify selected text command
- Verify clipboard content command
- Verification panel with results display
- Server management (start/stop/check)
- Status bar indicator
- Keyboard shortcuts
- Context menu integration
- Configuration options
- Risk level notifications

### Features
- Hallucination detection
- PII detection
- Security vulnerability scanning
- Harmful content detection
- Consistency analysis

### Privacy
- 100% local processing
- Zero telemetry
- No API keys required
